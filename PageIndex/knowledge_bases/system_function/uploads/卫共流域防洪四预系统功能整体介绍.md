# 卫共流域防洪“四预”能力提升系统介绍  

卫共流域是海河流域的重要组成部分,近年来洪涝灾害频发,三年两次启用蓄滞洪区,防洪形势严峻。本项目以多元数据底板为支撑,以调度模型为核心,以知识引擎为驱动,辅助物理流域实现“优调上游—滞蓄中游—集控下游”的防洪目标,提升卫共流域防洪“四预”能力。  

# 数据底板建设  

项目构建了全流域 L2 级数据底板和 $1440\mathrm{km}^{2}$ 重点区域 L3 级数据底板,形成了轻量化场景、高保真场景两套三维数字化场景,实现了天气、闸站、水情等各种要素的虚实交互。系统集成了流域内河流水系、水库、闸站、蓄滞洪区、水文测站、险工险段、南水北调等 7 类 5000 多项基础地理和工程数据,接入了水雨情监测、闸站监测、工情监测、生态监测、遥感监测、视频监测等 10 项 850 处实时监测数据,以及新建的 5 处无人机监控平台和 103 处三合一视频监控站点数据,实现了对卫共全流域实时水雨情的完全掌控,夯实了流域防洪的数据基础。  

# 二、模型库和知识库建设  

为满足防洪业务需要,项目构建了水文、水动力学、优化算法、时空数据分析等 4 类 127 个水利专业模型,通过自由耦合形成 10 类 30 个业务模型,支撑流域洪水预报调度、洪水正向和反向推演、河堤溃决、保留区分洪等情景预演,开发了模型服务平台,对各类模型、模型服务、模拟方案、计算资源等进行统一管理。  

通过对流域历史洪水、调度预案等知识提取挖掘,构建完成历史场景库、业务规则库等,通过复用省平台水利知识引擎结合卫共流域知识图谱,实现知识可视化、知识检索、知识问答、知识推荐等功能,  

# 三、防洪业务系统建设  

基于数据底板、模型库和知识库,开发了卫共流域防洪“四预”业务系统。  

(1)流域概况模块  

构建了全域二维高精地图和可交互式三维高保真场景,实现了河流水系、水利工程、防洪标准等 7 类 5000 多项基础和工程数据查询以及雨水情、工情、视频、遥感等实时信息查询,以及三维场景交互式漫游、数字孪生场景虚实交互等  

功能。  

(2)监测告警模块  

对流域范围内气象信息、实时雨情、水库和河道水情、闸站工情、工程安全、视频监测、遥感监测、AI 监测等实时信息的展示查询,当实时监测数据超过安全阈值时系统自动发出告警,主要告警项包括降雨、水库水位、河道水位、蓄滞洪区进洪、断面基流、AI 监测的人员车辆入侵等,所有告警信息系统自动整理后,可一键发送到相关人员手机。  

# (3)洪水预报模块  

通过耦合全流域 101 个子流域产汇流模型、主干河道一维河道水力学模型,基于中短期格网预测降雨,实现了每 6 小时一次的全流域洪水自动预报,预报结果包括大中型水库、河道断面、蓄滞洪区等,并可通过地图和图表等多种方式进行结果展示查询;系统也可进行灵活的人工洪水预报,通过手动输入降雨数据,可快速完成人工洪水预报方案的构建和计算,满足实际洪水预报需求。  

# (4)风险预判模块  

系统通过对当前洪水态势和未来水雨情发展态势的分析,结合工程实际,研判未来洪水和工程风险,包括水库高水位和泄洪风险,河道漫滩、漫堤、冲刷风险,蓄滞洪区进洪淹没风险等。系统提供包括水库纳雨能力分析、调洪演算分析,河道过流能力分析、滩地淹没分析,蓄滞洪区调洪能力分析等防洪能力分析工具。  

# (5)洪水预警模块  

系统基于洪水预报结果进行漫堤、冲刷、淹没等风险分析,对可能出现的洪水风险分类发出预警,包括水库洪水预警、河道洪水预警、蓄滞洪区进洪预警、南水北调中线预警等,所有预警信息在地图中突出展示,并可以短信一键发送至相关防汛人员手机。  

# (6)防洪预演模块  

1、正向推演:通过设定降雨洪水情景和闸站调度方案可对包括“21.7”洪水在内的各量级洪水进行洪水演进过程正向推演,全方位展示情景发生时水库、河道、蓄滞洪区洪水和淹没、冲刷等各类洪水风险情况,并可进行多方案对比分析;系统可自动结合洪水淹没情况进行避险转移路线、转移区和安置区规划和方案生成,并对受灾人口、面积、企业和经济损失进行详细的灾损评估;  

2、反向推演:系统可基于给定的降雨洪水情景和调度目标、约束条件,通过优化算法,反向推演水库、河道闸站和蓄滞洪区调度方案,并得到该调度方案下的洪水结果;通过同场洪水不同调度目标的对比、不同洪水相同调度目标的对比,可清晰展示各方向推演方案目标达成、调度方案和淹没损失情况。  

3、情景预演:在卫共主干河道范围内,可在任意堤防位置指定溃口进行溃堤模拟预演;针对大型水库,可进行任意洪水量级和溃决情景下的水库溃坝预演;针对主干河道拦河闸,可进行不同闸门故障下的风险情景预演;也可单独对河道洪水演进和各蓄滞洪区进洪进行模拟预演。  

# (7)防洪预案模块  

包括预案自动生成、关联预案管理等功能,根据预演情景的各预演方案,一键自动生成洪水风险防范预案,一键生成调度指令红头文件,并可在线预览和下载,为防洪决策、执行提供依据。  

# (8)防汛会商模块  

开发了丰富的防汛会商功能,包括调度预案管理、防汛人员物质管理、防汛值班、灾情现场直播、防汛综合会商等。
