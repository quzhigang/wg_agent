# 卫共流域防洪相关Java接口详细文档

## 概述

本文档描述了卫共数字孪生流域防洪预报预演系统的所有API接口，接口的输入输出参数均有完整描述。全部接口基于Spring Boot框架构建，使用玫瑰框架(Rose Kernel)作为基础框架。接口基础地址为：<http://10.20.2.153:8089>

## 接口分类概览

本文档共包含 **33个API接口**，按功能均为降雨相关，分类如下：

### 一、降雨预报接口（5个）

1. forecastRainEcmwf_avg - 获取流域平均的格网预报降雨过程
2. forecastRainEcmwf_each - 获取各子流域的格网预报降雨过程
3. forecastRainEcmwf_rect - 获取矩形区域内的格网预报降雨过程
4. forecastRainEcmwf_stc - 获取指定时段的降雨分区统计信息
5. forecastRainEcmwf_acc - 获取所有格网点指定时段的累计降雨

### 二、降雨等值面接口（7个）

6. contourRain_today - 获取8点以后降雨等值面
7. contourRain_any - 生成/获取任意时段累计降雨等值面
8. contourRain_plan - 生成/获取方案累计降雨等值面
9. contourRain_proc - 获取逐小时降雨等值面过程
10. contourRain_acc - 获取不同时段累计降雨等值面
11. contourRain_futureimg - 获取未来降雨等值面图片
12. contourRain_update - 更新等值面

### 三、降雨监测接口（2个）

13. monitorRainH_areaRainProcWhole - 获取指定时段流域整体面雨量过程
14. monitorRainH_manual - 手动更新降水监测数据

### 四、设计暴雨雨型管理接口（3个）

15. modelRainPattern_list - 查询设计雨型列表
16. modelRainPattern_add - 新增设计雨型
17. modelRainPattern_detail - 查看设计雨型详情

### 五、典型暴雨雨型管理接口（4个）

18. modelTypicalRain_list - 查询典型暴雨列表
19. modelTypicalRain_add - 新增典型暴雨
20. modelTypicalRain_detail - 查看典型暴雨详情
21. modelTypicalRain_addFromHistory - 从历史数据新增典型暴雨

### 六、方案降雨查询接口（7个）

22. modelRainArea_getByPlan - 获取指定方案的各子流域降雨过程
23. modelRainArea_getBasinAreaRainStc - 获取方案全流域平均面雨量过程及统计值
24. modelRainArea_getBasinAreaRainAcc - 获取方案全流域平均面雨量过程及实时累计降雨
25. modelRainArea_getBasinList - 获取方案有降雨预报的流域清单
26. modelRainArea_detail - 获取指定方案、指定流域的降雨过程及统计
27. modelRainArea_getByRsvr - 获取指定方案、指定水文站的上游流域降雨过程
28. modelRainArea_forecastRainStc - 获取自动预报方案的降雨态势

### 七、方案降雨设置接口（5个）

29. modelRainArea_addEcmwf - 根据格网预报设置方案降雨过程
30. modelRainArea_addEcmwfTranslate - 根据格网预报设置方案降雨过程（可放大平移）
31. modelRainArea_addManual - 手动设置方案降雨过程
32. modelRainArea_addManualCenter - 手动设置方案降雨过程（可设降雨中心）
33. modelRainArea_addBnd - 从数据库导入方案降雨过程

## 统一响应格式

所有接口返回统一的JSON格式：

```json
{
    "success": true,
    "code": "200",
    "message": "请求成功",
    "data": {}
}
```

错误时返回：

```json
{
    "success": false,
    "code": "500",
    "message": "错误信息",
    "data": null
}
```

## 附录

### A. 通用分页参数说明

所有分页查询接口支持以下参数:

- page: 页码，从1开始，默认1
- limit: 每页条数，默认10

### B. 日期格式说明

所有日期参数格式为: `yyyy-MM-dd HH:mm:ss`
示例: `2023-12-01 00:00:00`

### C. 状态码说明

- 200: 请求成功
- 500: 服务器内部错误
- 401: 未授权(需要登录)
- 403: 禁止访问

### D. 计算状态说明

模拟方案计算状态包括:

- 待计算
- 计算中
- 计算完成
- 计算失败
