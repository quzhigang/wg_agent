
## 文档搜索示例


PageIndex 目前默认支持在单个文档内进行基于推理的 RAG。
对于需要跨多个文档搜索的用户，我们针对不同场景提供了以下三种最佳实践工作流。

* [**按元数据搜索**:](metadata.md) 适用于可以通过元数据区分的文档。
* [**按语义搜索**:](semantics.md) 适用于具有不同语义内容或涵盖不同主题的文档。
* [**按描述搜索**:](description.md) 适用于少量文档的轻量级策略。
